はじめてのWebサイト制作 (午後)科 2022.5.16 - 11.15

仕事や自宅のパソコンで、Webサイトを閲覧するうちにいつしか自分でもWebサイトを作って見たいと思うようになりました。前職を退職後、転職活動をするなかで、Webサイト制作に関する仕事に就きたいと思い、求職者支援訓練を利用して、半年間に渡ってWebサイト制作を学びました。全くの未経験からスタートし、学ぶことも多く、大変ではありましたが、有意義な半年間を過ごすことができました。

Web Design

コーディング、レスポンシブ対応を想定したWebサイトを設計しています。どのデバイスでも綺麗に表示できるようなデザインを心がけ、ユーザーが使いやすく、わかりやすいWeb Designを目指しています。Web Designをするにあたっては、クライアントの要望に添うのはもちろんですが、将来的には、自分からもクライアントにより良い提案ができるようになりたいと考えています。

Coding

Web Designに基づき、HTML、CSSを用いてコーディングします。 keyframes、SVGアニメーションの表示やJavaScript、jQueryを使用したクリックイベントやスクロールイベント、プラグインを使用した動きのあるWebサイトを制作しました。また、WordPressでのテーマ、固定ページの作成やカスタムフィールドを使用したランキングのコントロールを行うことができます。

Graphic Design

Adobe Photoshop、Figmaを使用したデザインカンプの作成ができます。仮想クライアントワークでデザインカンプを作成する時には、クライアントとWebサイトの完成時のイメージを共有することを想定し、できるだけ細部まで作成しました。
 バナーを作成する時には、一目で何を目的としているバナーなのかが、わかるようなデザインを心がけました。
