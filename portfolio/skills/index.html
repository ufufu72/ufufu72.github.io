Skills

HTML5

HTML5の基本を理解し、適切なマークアップを行います。画像、リンクを表示したり、video要素を使うことで、動画や音声をシンプルに扱うことができます。picture要素を用いて、デバイスに合わせた画像を表示させることが可能です。フォーム関連では、history.backやautocomplete属性を使い、ユーザーがわかりやすく使いやすいフォームを作成することができます。

CSS3

CSS3の基本を理解し、関数、変数を使用することができます。flexboxを使用したレイアウトやobject-fitを使用した画像表示、vh、vw、%などの単位を指定することで、様々なデバイスに対応した記述を行うことができます。

Sass/SCSS

SCSS記法で記述します。@useを使用して、パーシャルファイルを読み込みます。関数、変数、@keyframesを使用したアニメーション、@mixinなどのメディアクエリの使用、ブレイクポイントの設定を行い、レスポンシブに対応します。コンパイラーを使用して、CSSを作成します。

Visual Studio Code

エディタはVisual Studio Codeを使用しています。HTML5、CSS3、Sass(SCSS)の記述、拡張機能の使用、GitHub、サーバー上のファイルの編集を行います。

JavaScript & jQuery

JavaScriptを使用して、年月日、時間の取得ができます。テキストデータを外部ファイルにして、Ajaxで読み込みます。if文の記述、clickイベント、scrollイベントを使用して、動きのあるWebサイト制作ができます。CDNを使用して、スライダー、ハンバーガーメニュー、アコーディオンメニューを実装することができます。

PHP & MySQL

XAMPPを使用して仮想サーバーを構築し、MySQLを使用して、データベースを構築します。PHPの基本的なコーディング、get/postを使用したデータ通信が行えます。PHPを使用して、年月日、時間を取得、includeでテンプレート化ができます。

WordPress

WordPressをダウンロードし、オリジナルテーマ、固定ページを作成し、ルートパスの記述ができます。カスタムフィールドを使用して、ランキングをコントロールすることができます。作成したファイルをサーバーにアップロードすることができます。現在、WordPressを使用した作品を制作しています。

git & GitHub

gitの基本設定を行い、GitHubと連動させ、ローカルにクローンを作成します。リポジトリ、README、ブランチ、issueの作成、ステージング、コミット、プッシュ、プルが行えます。GitHub Pagesを使用して、Webサイトを公開しています。
